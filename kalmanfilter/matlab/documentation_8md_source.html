1 # Discrete-discrete Extended Kalman filter
2 
3 Here we will describe the discrete discrete Extended Kalman filter that is of the square root flavour. Square root does not change the math behind the algorithm, but ensures that numerical rounding errors that are natural to a computer do not prevent the computed covariance matrices from being positive semi definite. For a linear model, the Extended Kalman Filter is equivalent to a regular Kalman filter.
4 
5 For the discrete-discrete Kalman filter the state evolves discretely and the measurements are discrete in time. The code is in `matlab_implementation/discrete`. The function header in `ddkef.m` is:
6 
7 
8 ```
9 function [estimates, covariances ] = ddekf(f_func,jacobian_func,dt_between_measurements,start_time,state_count,sensor_count,measurement_count,C,Q_root,R_root,P_0_root,x_0, measurements)
10 ```
11 
12 For those familiar with the Kalman filter and notation are familiar with the naming of the variables. However, to be extra sure it is always best to run `help ddekf`. We will break down an example below.
13 
14 ## Example 1
15 
16 The example is located in `matlab_implementation/discrete_discrete/examples/logistic.m`. First set the seed and include the file
17 
18 ~~~~~~~~~~~~~~~{.m}
19 clear all;
20 %include code
21 addpath('..')
22 
23 %setting the seed for repetition of experiment
24 isOctave = exist('OCTAVE_VERSION', 'builtin') ~= 0;
25 if isOctave==true
26  randn("seed",1);
27 else
28  rng(1);
29 end
30 ~~~~~~~~~~~~~~~
31 
32 Consider the nonlinear discrete logistic growth model.
33 
34 ~~~~~~~~~~~~~~~{.m}
35 %define discrete logistic growth model and its jacobian
36 rate = 0.01;
37 max_pop = 100;
38 next_func = @(x,t) x + rate.*x.*(1 - x/max_pop) + 0.*t;%time independent
39 jacobian_func = @(x,t) 1 + rate - (2*rate*x)/max_pop + 0.*t;
40 ~~~~~~~~~~~~~~~
41 
42 Set the time between each state evolution and the noise covariance and measurement matrices:
43 
44 ~~~~~~~~~~~~~~~{.m}
45 %in this model the time is only necessary
46 %for plotting an interpreting the results.
47 outputs = 1000;
48 t_start = 0;
49 t_final = 10;
50 dt = (t_final - t_start)/outputs;
51 
52 %discrete noise covariance
53 Q_d = 1;
54 %sensor noise covariance
55 R_d = 3;
56 %measurement matrix
57 C = 1;
58 
59 state_count = 1;
60 sensor_count = 1;
61 ~~~~~~~~~~~~~~~
62 
63 Set the initial estimates, covariances as well as generate data in order to run the filter:
64 
65 ~~~~~~~~~~~~~~~{.m}
66 %model results if there was no process noise
67 ideal_data = zeros(state_count,outputs);
68 %actual data (with process noise)
69 process_noise_data = zeros(state_count,outputs);
70 %noisy measurements of actual data
71 measurements = zeros(sensor_count,outputs);
72 
73 %initial estimate and covariance
74 %the initial estimate in this case
75 %matches the actual data
76 x_0 = max_pop/2;
77 P_0 = 1;
78 
79 %initial condition for ideal
80 %and real process data
81 x = x_0;
82 x_noise = x_0;
83 
84 %generate the data
85 for ii=1:outputs
86  x = next_func(x,ii);
87  ideal_data(:,ii) = x;
88 
89  x_noise = next_func(x_noise,0) + chol(Q_d)'*randn(state_count,1);
90  process_noise_data(:,ii) = x_noise;
91  measurements(ii) = C*x_noise + chol(R_d)'*randn(sensor_count,1);
92 end
93 ~~~~~~~~~~~~~~~
94 
95 Filter the data:
96 
97 ~~~~~~~~~~~~~~~{.m}
98 %filter the noisy measurements
99 [estimates, covariances] = ddekf(next_func,jacobian_func,dt,t_start,state_count,...
100  sensor_count,outputs,C,chol(Q_d)',chol(R_d)',chol(P_0)',x_0, measurements);
101 ~~~~~~~~~~~~~~~
102 
103 Plot the results:
104 
105 ~~~~~~~~~~~~~~~{.m}
106 %compare the measurements, estimates and true state.
107 %The ideal data is not plotted and is only for reference.
108 time = 0:dt:(outputs-1)*dt;
109 h = figure;
110 hold on;
111 plot(time,measurements);
112 plot(time,process_noise_data);
113 plot(time,estimates(1:end-1));
114 legend('Measurement','Real data','Estimate');
115 xlabel('Time')
116 ylabel('Population')
117 
118 h = figure;
119 hold on;
120 limit = 100;
121 plot(time(1:limit),measurements(1:limit),'LineWidth',2);
122 plot(time(1:limit),process_noise_data(1:limit),'LineWidth',2);
123 plot(time(1:limit),estimates(1:limit),'LineWidth',2);
124 legend('Measurement','Real data','Estimate');
125 xlabel('Time')
126 ylabel('Population')
127 ~~~~~~~~~~~~~~~
